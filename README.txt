Binvenue sur l'application WaveWatch 

Prérequis : Vous aurez besoin des logiciels suivants : wampserver (avec phpMyadmin) , VisualStudio code

Ce projet est une application de recomendation de séries a partir d'une base de soustitre 

Pour pouvoir l'installer il faut :

- Etape 1 : télécharger les fichier front 
- Etape 2 : mettre tout les fichier front dans le répertoire www de wampserver 
- etape 3 : éxécuter le fichier SQL correspondant a la base de données sur phpMyAdmin 
- Etape 4 : relier la base au projet (si cela n'est pas fait automatiquement)
- Etape 5 lancer le projet 

Voila maintenant vous pouvez noralement profiter de wavewatch.
En cas de problème veuiller contacter le concepteur de ce logiciel via ce mail : flomrian24@gmail.com

Les fichier "codes pythons" servent a créer la base de données mais consoment beaucoup de ressources sur votre ordinnateur donc il est conséiller d'utiliser la base de données importer via le fichier sql
Les fichier "codes pthons" servent uniquement pour un but de démontration et de compréhension du fonctionnement du logiciel
